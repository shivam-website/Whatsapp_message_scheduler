# WhatsApp Message Scheduler Bot

A Node.js bot that schedules and sends WhatsApp messages automatically using web-whatsapp.js.

## Features

- Schedule messages for future dates/times
- Beautiful web interface for management
- Real-time status monitoring
- Persistent storage of schedules
- Support for multiple phone numbers
- Automatic cleanup of sent messages
